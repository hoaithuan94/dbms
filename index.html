Summary:

Môn học giới thiệu các nội dung liên quan đến việc tổ chức, biểu diễn, cập nhật và khai thác thông tin trên hệ quản trị cơ sở dữ liệu quan hệ Microsoft SQL server. Nội dung chính gồm cách thức quản trị (tạo mới, cập nhật, tìm kiếm và truy vấn, xóa) cơ sở dữ liệu và các thành phần của nó (bảng dữ liệu, bảng ảo, bảng chỉ mục, thủ tục, v.v.) bằng các câu lệnh SQL và công cụ trực quan SQL Server Management Studio hoặc Azure Data Studio.
